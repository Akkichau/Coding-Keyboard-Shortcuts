# Kali Linux cheat sheet

### General
- F1 Online help
- F10 Left click menu on cursor
- Alt + F1 Open window
- Alt + F2 Open command window
- Alt + F4 Close window
- Alt + F7 Move window
- Alt + F8 Resize window
- Alt + F10 Toggle maximization state
- Ctrl + Alt + D Minimize all windows
- Alt + Space Activate the window menu
